# Open³

A public, auditable standard certifying that a project, product, or organization
is open across three independent axes — **source**, **data**, and **participation** —
against a fixed scoring rubric.

This repository is the standard itself: a single self-contained `index.html`
(no build step, no dependencies), served at **https://opencubed.org** via GitHub Pages.

## Editing

Edit `index.html` and push to `main`. GitHub Pages redeploys automatically.

- `CNAME` — binds the custom domain `opencubed.org`.
- The page is plain HTML/CSS with system light/dark; open it directly in a browser
  to preview.
